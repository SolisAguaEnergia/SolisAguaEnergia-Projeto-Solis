PROBLEMA
Em diversas comunidades rurais da Bahia, a falta de água potável ainda faz parte da realidade de muitas famílias.
Isso faz com que dependam de fontes contaminadas ou caminhões-pipa, aumentando o risco de doenças.
Além disso, a falta de infraestrutura agrava ainda mais essa situação.

SOLUÇÃO
Para enfrentar esse problema, desenvolvemos uma solução sustentável e acessível.
O projeto propõe a construção de poços artesianos, utilizando energia solar para o bombeamento da água.
Com apoio de parceiros, conseguimos levar água limpa, gratuita e contínua para as comunidades.

ODS

ODS 7: busca assegurar o acesso confiável, sustentável, moderno e a preço acessível à energia para todos 
ODS 11: busca tornar as cidades e os assentamentos humanos inclusivos, seguros, resilientes e sustentáveis.
ODS 12: busca assegurar padrões de consumo e produção sustentáveis, promovendo a eficiência no uso de recursos naturais e a redução do desperdício.
ODS 17: busca fortalecer a parceria global para o desenvolvimento sustentável, promovendo a colaboração entre países, organizações e setores

O site institucional do projeto oferece as seguintes ferramentas:

1. Visualização de Impacto: Dashboard com números reais (famílias atendidas, poços construídos e comunidades beneficiadas).
2. Mapeamento de Atuação: Carrossel interativo mostrando as cidades atendidas (Rodelas, Cairu, Coronel João Sá, Carinhanha, Monte Santo, Campo Formoso e Jaguarari).
3. Área Restrita: Sistema de login simulado para voluntários e parceiros cadastrados.
4. Captação de Recursos: Páginas dedicadas para doações financeiras e propostas de parcerias corporativas.
5. Gestão de Voluntários: Formulário de cadastro para novos colaboradores com seleção de disponibilidade e área de atuação.

Tecnologias:
O projeto foi desenvolvido utilizando tecnologias modernas de front-end, focando em performance e acessibilidade:

• HTML5: Estrutura semântica para melhor SEO e acessibilidade.
• CSS3: Estilização avançada com uso de Flexbox, Grid e animações customizadas (como o efeito de "onda" e transições de fade-up).
• JavaScript (Vanilla): Lógica de interatividade, carrossel de imagens, validação de formulários e manipulação do DOM sem dependências pesadas.

Como rodar o Projeto:
omo o projeto é composto por arquivos estáticos (Frontend), o processo de execução é simples:

1.Clonar/Baixar o Repositório:
Certifique-se de ter todos os arquivos e pastas (css, js, assets, pages) na mesma estrutura de diretórios.

2.Execução Local:
•Basta abrir o arquivo index.html em qualquer navegador moderno (Chrome, Firefox, Edge).
•Para uma experiência melhor (evitando problemas de caminhos relativos em alguns navegadores), recomenda-se usar a extensão Live Server do VS Code ou qualquer servidor HTTP simples.
